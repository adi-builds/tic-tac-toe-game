# ❌⭕ Tic Tac Toe

A modern and responsive **Tic Tac Toe game** built using **React** and **Tailwind CSS**. Play against a friend and enjoy a clean UI with smooth interactions.

---

## 🚀 Features

* 🎮 Two-player gameplay (X vs O)
* 🔁 Restart / Reset game
* 🧠 Win detection logic
* 🤝 Draw detection
* ⚡ Fast and responsive UI
* 🎨 Styled with Tailwind CSS

---

## 🛠️ Tech Stack

* React (Vite)
* JavaScript (ES6+)
* Tailwind CSS

---

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/adi-builds/tic-tac-toe-game
```

2. Navigate to the project folder

```bash
cd tic-tac-toe
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

---

## 🎮 How to Play

* Player **X** starts first
* Players take turns placing marks
* First to align 3 marks (row, column, diagonal) wins
* If all cells are filled → it’s a draw
